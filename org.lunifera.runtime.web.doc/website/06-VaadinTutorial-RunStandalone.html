Vaadin tutorial - "Run the standalone example"

  • Import projects
  • Set target platform
  • Launch application
  • Run example
  • Further Informations

Running the addon example is really easy. You only have to download the example from vaadin addons. It contains all information required.

ATTENTION: No vaadin push is provided by the example!

Top

Import projects

Follow this http://vaadin.com/directory to download the example.

Please unzip the jar file and import the project into your workspace.

Image 1: Import project into workspace

Top

Set target platform

Set the target platform - Open org.lunifera.examples.runtime.web.vaadin.standalone.target and wait until target platform is resolved. If problems occur, please select the problematic repo and use the update button to refresh versions. If target was resolved, click "Set as Target Platform".

Image 2: Set the target platform

Top

Launch application

Open launch configuration "vaadinstandalone" and ensure that all bundles are available. Use button "Validate" and "Add Required Bundles".

Following bundles have to be set to autostart:

  • org.lunifera.runtime.component.configuration.manager
  • org.eclipse.equinox.cm
  • org.eclipse.equinox.ds
