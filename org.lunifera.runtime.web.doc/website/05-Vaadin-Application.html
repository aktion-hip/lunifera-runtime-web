  • Test

The bundle org.lunifera.runtime.web.jetty was introduced to provide the capability in starting / stopping and controlling different instances of jetty servers. Each of this instances should be independent from each other.

  • The main issue was to use CM (OSGi Configuration manager) to handle all the settings required to setup an instance of jetty properly.
  • Another important issue was to capability of providing Jetty-Handlers from other bundles to an configured instance by OSGi services.
